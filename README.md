# BackendChat
#### Описание:
Разработана  ___API___ чатов и сообщений.

#### Функциональность :

- Создание чата.

- Отправка сообщения в чат.

- Получение чата с последними N сообщениями.

- Удаление чата со всеми сообщениями.

#### Технологии и инструменты:

- ___Python 3.14.2___

- ___FastAPI___

- ___PostgreSQL___

- ___SQlalchemy___

- ___alembic___

- ___Docker___

- Асинхронная обработка запросов

### Запуск
-------------
#### Клонировать репозиторий
```bash
git clone https://github.com/Baranochka/BackendChat.git
```

#### Перейти в директорию
```bash
cd BackendChat
```

#### Запустить docker-compose
```bash
docker compose up --build
```
