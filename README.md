# BackendChat
#### Описание:
Разработана  ___API___ чатов и сообщений.

#### Функциональность :

- Создание чата.

- Отправка сообщения в чат.

- Получение чат с последними N сообщениями.

- Удаление чата со всеми сообщениями.

#### Технологии и инструменты:

- ___Python 3.14.2___

- ___FastAPI___

- ___PostgreSQL___

- ___SQlalchemy___

- ___alembic___

- ___Docker___

- Ассинхронная обработка запросов
